# 발스리그 개발 히스토리

## v1.0 — 로그인 / 회원가입

- Clerk 기반 로그인·회원가입 페이지 구현 (`/sign-in`, `/sign-up`)
- Convex DB에 로그인 정보 및 개인정보 저장
- 회원가입 후 프로필 수정 페이지에서 개인정보(클랜소속·이름·닉네임) 저장
- 로그인 후 `/leagues`로 자동 이동

## v1.3 — 순위표
