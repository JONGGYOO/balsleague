1. 로그인 페이지 작성
 - Clerk을 이용해서 먼저 로그인 페이지를 만들어 볼게. 
 - convex DB에 로그인 정보 및 개인정보를 저장하게 할거야
 - 회원가입을 하고 수정 페이지에서 개인정보를 저장 할거야
 - 개인정보는 클랙소속, 이름, 닉네임 필드를 입력하고 수정하도록 할거야.

2. 회원 가입을 한 인원이 축구게임으로 리그를 진행하면서 스코어를 입력하는 프로그램을 작성할거야
   우선 리그 리스트를 만들고 수정하는 화면을 작성할게
 - 로그인 하고 메인화면에서는 진행중인 리그 전체 리스트가 나오게 할거야.
 - 그리고 리그를 등록할거야. 리그는 년도별,월별로 등록할거고 텍스트로 리그명도 입력하게 될꺼야.
 
 
3. 스코어 입력
 - 우선 사용자는 해당리그에 참가 버튼을 눌러서 참가/취소 버튼으로 제어할꺼야 
 - 스코어입력을 할 수 있도록 하는데 참가자를 셀렉트박스로 전체 보여주고 나와 스코어를 입력 할 수 있도록 해줘. 그리고 셀렉트박스는 첫글자를 입력하고 이후 글자를 입력 할때마다 글자 마다 보여지는 참가자를 줄여줘.

4. 순위표
 - 입력된 스코어를 토대로 순위표를 만들어줘. 승무패가 보여지고 득실의 표현해줘.
 순위 소트는 승무패 득실의 순으로 표현해줘. 동일일때는 경기를 많이 한 사람이 순위가 높아.

5. 권한설정

6. 경기결과 리스트 표시
