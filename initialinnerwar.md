1. 내전 항목 추가
  .1 우선 리그 추가에서 구분을 하나 더 넣을께 리그/내전을 선택할 수 있게 셀렉트 박스를 넣고 저장하게 해줘
  .2 내전을 선택할때는 년도/월/일까지 선택하고 저장하게 해줘
2. 내전 페이지 (리그는 기존대로 하고 내전 페이지를 하나 더 만들께)
  .1 내전도 동일하게 참가신청과 참가취소, 수정 삭제로 만들어줘
  .2 내전을 눌렀을때 페이지는 내전상세 라고 할께
  .3 내전 상세 페이지에서는 참가자를 대상으로 A팀과 B팀으로 나눌꺼야
  .4 팀을 나누는건 차후에 리그 기록이나 내전 승리를 참고해서 나누도록 하는 '성적기반' 버튼과 '랜덤버튼'으로 나누도록 할거야
  .5 그리고 슈퍼관리자 와 관리자가 직접 선택해서 A팀 B팀으로 나눌수도 있도록 해줘
  .6 A팀 B팀이 뽑히면 나열되어 서로 경기를 하면서 이기는 사람이 계속 위로 올라 갈거야 스코어를 입력하게 해서 승자가 계속 게임을 하도록 해줘
  .7 양쪽에서 경기를 하다가 모든 선수가 지게 되면 그 팀은 패배가 되고 경기는 끝나. 경기가 끝나면 승리한 팀에게 축하 메세지를 줘.
  .8 전체 경기 결과를 리스트로 표시해주고 그리고 개인별 리스트에 승리 수와 패 득점 실점도 표시해줘
  1. 경기
    -1 내전은 승인 절차 없이 바로 참가 하도록 수정해줘.
    -2 플레이어가 직접 순번을 변경 할 수 있도록 해줘
    -3 경기 시작 버튼을 만들어줘. 경기 시작을 하면 팀 변경을 할 수 없고 이후에 점수 입력 할 수 있게 해줘.
    -4 점수 입력 하고 다음 입력 전까지 입력한건 수정 할 수 있게 해줘.
3. 경기 방식 추가
  -1 동점시 진행 중일때는 동반 탈락하고 각팀별로 다음 선수 점수 등록하도록하고
  지막 경기는 동점 허용하지 않는다. 그리고 마지막 두선수 일때는 알림으로 연장이나 승부차기 셋팅을 하세요라는 알림 표시해준다.
  -2 팀 배정 초기화 (재시작)은 표시는 해주고 슈퍼관리자와 관리자만 클릭 할 수 있다. 
  -3  슈퍼관리자 와 관리자가 직접 선택해서 A팀 B팀으로 나눌수도 있도록 해달라고 했는데 
  전 추가시에 이부분을 셋팅을 할께 팀 배분을 관리자만 할수 있는지 일반사용자도 할 수 있는지 선택을 해서 해당 권한 별로 A팀 B팀으로 나눌수도 있고 '성적기반' 버튼과 '랜덤버튼'도 누룰수 있게 해줘
  기시작 버튼은 모두가 누를수 있도록 해줘. 여기에 초기화 버튼도 같은 권한 설정해줘
  -4 순서 변경도 모든 사용자가 가능하도록 해줘
4. 경기 방식 추가
  -1 경기 시작후 초기화 버튼을 누르면 관리자만 초기화 할 수 있는 상태라고 해줘
  -2 A팀 B팀 셋팅이 완료되면 램덤 배정 버튼을 누를때와 마찬가지로 화면 구성을 해줘 지금은 화면이 달라서 순서 정하는데 화면이 보기 힘들어
  -3 점수 입력은 본인만 할 수 있는데 슈퍼관리자나 관리자는 꼭 본인 경기가 아니어도 점수 입력을 할 수 있게 해줘
5. 점수입력권한

-1 점수 입력은 슈퍼관리자와 관리자가 가능하지만 본인 스코어는 양쪽 경기자 중에 둘다 입력 가능하게 해줘

1. 내기 전달 화면 개발  
-1 내전 추가에 내기품목 텍스트로 입력 할 수 있게 추가해줘  
-2 팀 승리 완료 후 처음 선수 순번에서 패전 팀이 승리 팀에게 순서대로 내기품목 전달이라는 화면을 만들어줘  
-3 기존 내전은 모두 내기품목에 메가아아 를 입력해서 보여줘
